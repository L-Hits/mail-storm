# 📧 Mail Storm 📧

> AI를 이용한 메일의 스팸 여부 판단 사이트! 👉 https://vactor0911.github.io/mail-storm/

![사용 예시 이미지]()

## 📖 Description

메일의 제목을 입력하여 해당 메일의 스팸 여부를 예측하는 서비스입니다.

학습된 SVM 인공지능 모델을 활용해 메일의 제목으로 스팸 여부를 예측합니다.
